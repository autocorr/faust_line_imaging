FAUST Archival Pipeline
=======================
Welcome to the documentation for the FAUST Archival Imaging Pipeline! These web
pages contain information on how to generate deconvolved image products for the
ALMA `FAUST <http://faust-alma.riken.jp/>`_ Large Program. For instructions on
how to setup your host environment in order to use the pipeline see the
:doc:`environment <environment>` page. Once all datasets and paths are in the
appropriate locations, please refer to the :doc:`quick-start guide
<quickstart>` for information on executing pipeline jobs. Information for
running specialized jobs for custom imaging may be found under the
:doc:`recipes <recipes>` page.

The pipeline is authored by Brian Svoboda and Claire Chandler. The code is
released under the MIT License. A copy of the license is supplied under the
LICENSE file distributed with the software. The code may be found on `GitHub
<https://github.com/autocorr/faust_line_imaging>`_.
